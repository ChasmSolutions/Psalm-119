Add an encouraging exerpt from Psalm 119 to the admin section of your Wordpress. 

== Description ==

This plugin adds an exerpt from Psalm 119 to the upper content area of the admin section of Wordpress. Based on Matt Mullenweg's 'Hello Dolly' plugin.

== Installation ==

Install the plugin through the Plugins screen. No configuration neccissary.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
